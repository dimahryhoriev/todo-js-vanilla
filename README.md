# todo-js-vanilla
Get-It-Done: A simple and functional
To-Do List web application.

[🟢 Live Demo](https://dimahryhoriev.github.io/todo-js-vanilla/)

🛠 Tech Stack
  * JavaScript: Vanilla JS (DOM manipulation, LocalStorage)
  * Styles: SCSS (BEM methodology, Flexbox, Dark/Light mode)
  * HTML: Semantic markup, HTML5 Templates

⚡️ Key Features
  * Persistence: Tasks are saved in localStorage
  * Filtering: Easily switch between All, Active, and Completed tasks
  * Themes: Smooth dark/light mode toggle with preference memory
  * Responsive: Fully optimized for mobile and desktop screens


🚀 How to Run

-- Option 1: Local Setup
  * Clone the repository:
   `git clone https://github.com/dimahryhoriev/todo-js-vanilla.git`
  * Open `index.html` in your browser.

-- Option 2: Instant View
  * Just open the [Live Demo](https://dimahryhoriev.github.io/todo-js-vanilla/) in your browser.
